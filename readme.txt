Please find below a description of what each file is:


AwesomeApp - C# Project, the mobile application itself


MachineLearningFilesFolder - contains the following:
FIN_dataset.csv - downloaded directly from the Google Forms survey. This is the raw data
mydata5.csv - cleaned data. This is used in the machine learning
MachineLearningCode - R code for machine learning
glm.pmml, rpart.pmml, nb.pmml - pmml files which contains the machine learning models.
R-shiny_FIN_Application - code and dataset for the R-Shiny App. This is not implemented in the final version of the application
